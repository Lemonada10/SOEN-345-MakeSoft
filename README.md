# SOEN-345-MakeSoft
Cloud-based Ticket Reservation Application

# run backend
- cd project
- mvn spring-boot:run

# run frontend
- cd frontend
- npm install
- npm start

# Deployment tools
- Vercel (frontend/ auto updates)
- Render (backend/ need to redeploy after each backend change/commit)
